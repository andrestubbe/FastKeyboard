# Changelog

All notable changes to **FastKeyboard** will be documented in this file.

## [0.2.0] - 2026-05-16

### Added
- **v0.2.0**: High-precision timestamps (`GetMessageTime`) for all keyboard events.
- **v0.2.0**: Real-time character translation (`ToUnicode`) passed directly to Java.
- **Roadmap**: Added `ROADMAP.md` for project vision.
- **Bat Scripts**: Improved `run-demo.bat` with quiet mode and auto-install.

### Fixed
- Demo project dependency issues and missing `pom.xml`.

## [0.1.0] - 2026-05-16

### Added
- Initial public release.
- Native Win32 RawInput support.
- Background interception capabilities.
- JitPack support with bundled DLL.
